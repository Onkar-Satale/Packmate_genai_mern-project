# Codathon PackMate

PackMate is a **smart packing assistant** built with React. It helps users **organize and optimize their travel packing** using AI.

## 🚀 Features
- AI-powered packing suggestions
- Weather-aware recommendations
- Customizable travel lists
- Easy-to-use interface

## 🧠 Tech Stack
- **Frontend:** React, HTML, CSS, JavaScript
- **Backend:** Node.js
- **APIs:** OpenCage API (location), Gemini API (AI suggestions)
